视觉风格规范 v1

这是 v2 的设计令牌（design token）真相源与核心组件样例。研发取色/取间距/取圆角一律引此处的 CSS 变量，不再硬编码。本文件自身即用这套 token 渲染——所见即所得。

基调（已定）：亮色默认 · 暗色一键可切 · 柔和现代（大圆角 / 充足留白 / 0.5px 细边） · 系统字体栈（中文原生黑体 + Inter 拉丁）。锁定项见规格 §17。点右上角切换明暗，下方所有色值实时更新。

1 · 调色板

§17 中性黑白 + 陶土点缀

中性面是微暖中性灰（不是纯灰），与陶土点缀同源、更耐看。主操作恒为黑（暗色下反相为浅）。陶土 #C26A3D 是全站唯一强调色，只用于「推荐」徽章 / 选中点睛 / 链接 hover，绝不大面积铺。

10 · 灵感卡（封面为主体 · 瀑布流自适应）

§13

封面图为视觉主体、尽量大；标题/摘要/操作以半透明渐变浮层叠在图片下半部，上半部图片完全可见。品类标签浮左上、"用此提示词" hover 浮现（陶土点睛）。封面保留图片原始比例（瀑布流），整图不裁切——下方 2:3 / 1:1 / 16:9 / 3:2 四种比例同墙自适应（真实数据把占位换成 <img>、并把该图宽高比写进 .cov 的 aspect-ratio 即可）。

瀑布流（CSS columns，宽度收窄自动 3→2→1 列）+ 每卡按图片真实比例设 aspect-ratio，故整图可见不裁切。浮层文字/标签固定白色（永远叠在不可预测的图上，保证可读），卡片外框/底/阴影仍走 token 随明暗切换。

12 · 用法约定 / 红线

落地必守
陶土 = 点睛，不是主色。全站陶土只出现在 4 处：① 「推荐 / 更划算」徽章 ② 选中 / 激活的细节点睛 ③ 链接 hover ④ 推荐档卡片描边。主操作、大面积背景一律中性。
  • 取色 / 间距 / 圆角一律用 CSS 变量，不硬编码（取代 v1 styles.css 的散落色值）。
  • 层级用 0.5px 细边 + 微阴影界定，不用重边框、不用大投影。
  • 主操作按钮恒 --primary-bg（亮黑 / 暗浅自动反相），保证最高对比。
  • 暗色不是把亮色取反——已单独调校（陶土提亮、语义色降饱和、阴影加深）。
  • 圆角默认：卡片 lg(16)、输入/下拉 md(12)、药丸/按钮/发送 full。
  • 所有图片可点击放大（lightbox），浮层内仅下载。

确认本风格后：① 同步进规格 §17（把抽象描述替换为本 token 表）② 更新 PROGRESS / 原型 README 索引 ③ 阶段一开发时把这套变量落成 tokens.css 全局引入。
