# 原型（低保真线框）

后续研发"照着原型开发"的入口。**结构看这里的图，行为/规则看 [redesign-requirements.md](../redesign-requirements.md)。**

## 怎么看
- **结构/布局** → 浏览器打开 **[wireframes.html](wireframes.html)**（单文件、自带样式，联网时图标走 CDN；离线也能看布局）。一页里从上到下排了全部页面与状态，每节标题旁标了对应的需求章节（§x）。
- **视觉风格/设计令牌** → 浏览器打开 **[design-system.html](design-system.html)**（已定稿：明暗两套、全部 token + 组件样例 + 用法红线；右上角可切明暗。研发取色/间距/圆角一律引其 CSS 变量）。

## 这是什么 / 不是什么
- **wireframes.html 是**：页面有哪些、布局怎么排、组件层级、各状态长什么样（中性灰块占位，不表达最终视觉）。
- **最终视觉风格已定稿** → 见 **[design-system.html](design-system.html)**（配色/字体/圆角/暖色点缀/组件样例，明暗两套）。wireframes 看结构、design-system 看风格、redesign-requirements.md 看行为规则。
